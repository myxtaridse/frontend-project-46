# Gendiff

**Gendiff** is a command-line utility that finds the differences between two data structures. It supports **JSON** and **YAML** formats and outputs the result in a clean, readable format. This tool is useful for debugging configurations and comparing nested data.

## For install Gendiff:
```bash
git clone https://github.com/myxtaridse/frontend-project-46.git
cd frontend-project-46
make setup
```

## Usage (to compare two file):
```bash
gendiff <filepath1> <filepath2>
```
